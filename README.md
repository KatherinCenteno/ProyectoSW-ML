  (iii) JavaScript <br>

### Back-End

## Key Features
### Public User
(i) Search Product <br>
(ii) View Product <br>
(iii) Create User Account <br>

### Signin User
(i) Search Product <br>
(ii) View Product <br>
(iii) Create Order <br>


### Admin
(i) Add New Product <br>
(ii) Update Product <br>
(iii) See all Orders <br>
(iv) Manage all Users <br>
